## Exercice 1 : Analyse d’un nombre entier (10 points)

Écrire un programme en C qui :

   * Si le nombre est invalide, afficher un message d’erreur.
3. Si le nombre est valide, le programme doit :

   * indiquer si le nombre est **pair ou impair**,

### Exemple d’affichage attendu (à titre indicatif)

```
Entrez un nombre : 6
Le nombre est pair
Les nombres de 1 à 6 sont :
1 2 3 4 5 6
```

## Exercice 2 : Évaluation de notes sur plusieurs étudiants (10 points)

On souhaite écrire un programme qui évalue les notes de plusieurs étudiants.

### Travail demandé

2. À l’aide d’une **boucle**, pour chaque étudiant :

   * demander une **note comprise entre 0 et 20**,
   * vérifier la validité de la note,
   * afficher l’appréciation correspondante :

| Intervalle | Appréciation |
| ---------- | ------------ |
| 0 – 9      | Insuffisant  |
| 10 – 12    | Moyen        |
| 13 – 15    | Bien         |
| 16 – 18    | Très bien    |
| 19 – 20    | Excellent    |

3. À la fin du programme, afficher :

   * le nombre total d’étudiants qui ont la mention Bien.
